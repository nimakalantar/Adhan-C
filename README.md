# Adhan C

Adhan C is written to be compatible with embedded systems and microcontrollers.

## Usage

### Building

```bash
rm -rf build
cmake -S . -B build
cmake --build build
```

### Run unit tests

```bash
./build/test/runUnitTests
```

### Run example program

```bash
./build/src/example
```
